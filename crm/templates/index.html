About CLASER

Sistem Claser (Clustering Customer) merupakan aplikasi berbasis website yang berfungsi untuk memberikan rekomendasi kelompok pelanggan berdasarkan kesamaan karakteristiknya. Karakteristik kelompok pelanggan menjadi isu penting bagi perusahaan dalam menentukan strategi pemasaran yang tepat sasaran.

Model data yang digunakan pada sistem ini adalah model data RFM dan LRFM dengan pembobotan model data menggunakan pembobotan Analytic Hierarchy Process (AHP).

Proses clustering dilakukan dengan menggunakan metode K-Means dan dilanjutkan dengan penentuan peringkat cluster pelanggan dengan menggunakan metode Technique for Order Preference by Similarity to Ideal Solution (TOPSIS).

Tahapan CLASER

Input Data

Input yang dibutuhkan pada sistem ini adalah data RFM atau LRFM, jumlah k cluster. Selain itu penilain terhadap atribut model data juga dapat diberikan oleh pengguna namun optional karena terdapat bobot default pada sistem.

Proses

Data RFM atau LRFM dinormalisasi menggunakan MinMax Normalization. Penilain yang diberikan terhadap atribut model data diproses dengan AHP untuk menghasilkan bobot. Data yang telah dinormalisasi kemudian dikalikan dengan masing-masing bobot data. Selanjutnya proses clustering dilakukan terhadap data terbobot sebanyak jumlah k cluster. Objek data pada setiap cluster dihitung nilai Silhouette Coefficientnya untuk mengetahui kualitas cluster. Centroid dari hasil clustering terbaik diberikan rangking menggunakan TOPSIS dan pengujian Rank Consistency untuk mengetahui rekomendasi cluster terbaik. Terakhir evaluasi dilakukan terhadap karakteristik cluster.

Output Hasil

Hasil akhirnya adalah rekomendasi cluster pelanggan beserta karakteristiknya disertai dengan saran startegi pemasaran.
